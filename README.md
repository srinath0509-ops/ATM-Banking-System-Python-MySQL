# ATM Banking System using Python & MySQL

## Overview
A console-based ATM Banking System developed using Python and MySQL. The application simulates real-world banking operations such as account creation, login authentication, balance inquiry, deposits, withdrawals, fund transfers, and transaction history management.

## Features
- Create Account
- Secure PIN Authentication
- Balance Inquiry
- Deposit Money
- Withdraw Money
- Transfer Funds
- Mini Statement
- Transaction Logging
- MySQL Database Integration

## Technologies Used
- Python
- MySQL
- MySQL Connector
